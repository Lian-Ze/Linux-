# 一、Linux 远程登录

## 1. 什么是 Linux 远程登录？

Linux 远程登录是指：

> 在一台计算机上，通过网络连接并操作另一台 Linux 计算机。

连接结构：

```text
Windows电脑
    ↓
网络
    ↓
Linux服务器
```

远程登录后，可以在本地电脑的终端中执行 Linux 命令，就像直接操作服务器一样。

---

## 2. Linux 远程登录的应用场景

在公司进行软件开发时，通常不会让每一名开发人员都单独使用一台服务器。

更常见的情况是：

```text
开发人员A
开发人员B
开发人员C
    ↓
共享开发服务器
```

开发小组成员可以通过远程登录的方式，共同使用服务器上的：

- 代码
- 编译环境
- 数据集
- 测试工具
- 计算资源

在项目正式上线后，程序通常会运行在公网服务器上。

应用场景可以分为：

### 开发阶段

```text
本地电脑
    ↓ SSH远程登录
公司内部Linux开发服务器
```

### 项目上线阶段

```text
本地电脑
    ↓ SSH远程登录
公网Linux服务器
    ↓
运行正式项目
```

---

## 3. Linux 远程登录的重要性

Linux 远程登录是服务器开发中的基础技能。

以后学习以下内容时都会经常使用：

- Linux 服务器
- C/C++ 项目开发
- 网络编程
- Git 代码部署
- OpenCV 项目运行
- 深度学习模型部署
- ROS2 机器人系统
- 云服务器项目部署

对于未来的嵌入式 AI 和机器人方向，远程登录可以用于：

- 登录机器人主机
- 查看程序运行状态
- 修改配置文件
- 调试程序
- 查看系统日志
- 上传和运行代码

# 二、Xshell 工具

## 1. Xshell 是什么？

Xshell 是一款用于远程登录 Linux 服务器的终端工具。

它主要通过 SSH 协议连接 Linux 服务器。

连接结构：

```text
Windows电脑
    ↓ Xshell
SSH协议
    ↓
Linux服务器
```

---

## 2. Xshell 的主要作用

使用 Xshell 可以：

- 远程登录 Linux 服务器
- 执行 Linux 命令
- 编译和运行程序
- 修改配置文件
- 查看系统状态
- 管理服务器文件
- 进行项目调试

连接 Linux 服务器时通常需要填写：

- 服务器 IP 地址
- SSH 端口号
- Linux 用户名
- Linux 用户密码

SSH 默认端口通常是：

```text
22
```

---

## 3. Xshell 的基本使用流程

使用 Xshell 连接 Linux 服务器的一般流程：

1. 获取 Linux 服务器的 IP 地址。
2. 打开 Xshell。
3. 创建新的连接。
4. 选择 SSH 协议。
5. 输入 Linux 服务器 IP 地址。
6. 输入端口号，通常为 `22`。
7. 输入 Linux 用户名和密码。
8. 连接成功后进入 Linux 终端。

连接成功后，可以在 Xshell 中输入以下命令进行测试：

```bash
pwd
ls
whoami
uname -a
```

# 三、XFTP 工具

## 1. XFTP 是什么？

XFTP 是一款用于 Windows 和 Linux 之间传输文件的工具。

它通常使用 SFTP 协议进行文件传输。

文件传输结构：

```text
Windows电脑
    ↓ XFTP
SFTP协议
    ↓
Linux服务器
```

---

## 2. XFTP 的主要作用

使用 XFTP 可以：

- 将 Windows 中的代码上传到 Linux 服务器
- 将 Linux 服务器中的文件下载到 Windows
- 上传项目资源
- 下载运行日志
- 管理服务器目录
- 传输图片、数据集和配置文件

XFTP 通常采用左右两侧的窗口：

```text
左侧：Windows本地文件
右侧：Linux服务器文件
```

可以通过拖拽的方式完成文件上传和下载。

---

## 3. Xshell 与 XFTP 的区别

| 工具 | 主要作用 |
| --- | --- |
| Xshell | 远程登录 Linux 并执行命令 |
| XFTP | 在 Windows 和 Linux 之间传输文件 |

二者可以配合使用：

```text
XFTP：上传项目代码
    ↓
Xshell：登录 Linux 服务器
    ↓
编译、运行和调试项目
```

# 四、Linux 文本编辑器 vi 和 vim

## 1. vi 和 vim 是什么？

vi 和 vim 是 Linux 中常用的文本编辑器。

它们可以用于：

- 编辑文本文件
- 编写 Shell 脚本
- 修改配置文件
- 编写 C/C++ 代码
- 修改服务器文件
- 查看和编辑程序代码

vim 可以看作是 vi 的增强版本，功能比 vi 更加丰富。

---

## 2. vi 和 vim 的区别

| 编辑器 | 说明 |
| --- | --- |
| vi | Linux 中经典的文本编辑器 |
| vim | vi 的增强版本，功能更加丰富 |

在实际开发中，vim 使用得更加广泛。

打开或创建文件：

```bash
vim test.txt
```

如果系统中没有 vim，也可以尝试使用：

```bash
vi test.txt
```

# 五、vim 的三种模式

vim 主要有三种模式：

1. 正常模式
2. 插入模式
3. 命令行模式

模式切换关系：

```text
        按 i、a、r
正常模式 ─────────→ 插入模式
    ↑                  │
    │      按 Esc      │
    └──────────────────┘

正常模式 ──输入 : 或 /──→ 命令行模式
```

---

## 1. 正常模式

在终端中输入：

```bash
vim 文件名
```

进入 vim 后，默认处于正常模式。

正常模式主要用于：

- 移动光标
- 复制文本
- 粘贴文本
- 删除文本
- 撤销操作
- 查找内容
- 跳转到指定位置

注意：

> 只有在正常模式下输入快捷键，才会执行对应的编辑功能。

---

## 2. 插入模式

在正常模式下按下以下按键，可以进入插入模式：

```text
i
a
r
```

其中最常用的是：

```text
i
```

进入插入模式后，就可以输入和编辑文本。

### 常用按键

| 按键 | 作用 |
| --- | --- |
| `i` | 在光标当前位置前插入文本 |
| `a` | 在光标当前位置后插入文本 |
| `r` | 替换当前光标所在的一个字符 |
| `Esc` | 退出插入模式，返回正常模式 |

---

## 3. 命令行模式

在正常模式下输入：

```text
:
```

可以进入命令行模式。

输入：

```text
/
```

可以进入搜索模式，用于查找文件中的内容。

命令行模式常用于：

- 保存文件
- 退出 vim
- 强制退出
- 查找内容
- 设置行号
- 取消行号

---

# 六、vim 的保存与退出

在正常模式下输入以下命令：

## 1. 保存并退出

```text
:wq
```

其中：

```text
w = write，保存
q = quit，退出
```

操作步骤：

1. 按下 `Esc` 返回正常模式。
2. 输入 `:wq`。
3. 按下回车。

---

## 2. 退出文件

```text
:q
```

作用：

> 退出 vim。

如果文件没有被修改，可以直接使用该命令退出。

---

## 3. 强制退出且不保存

```text
:q!
```

作用：

> 强制退出，并且不保存修改内容。

---

## 4. 只保存不退出

```text
:w
```

作用：

> 保存文件，但不退出 vim。

---

# 七、vim 常用快捷键

以下快捷键需要在正常模式下使用。

## 1. 复制

```text
yy
```

作用：

> 复制光标所在的一行。

复制多行：

```text
5yy
```

作用：

> 从当前光标所在行开始，复制下面五行。

---

## 2. 粘贴

```text
p
```

作用：

> 将复制的内容粘贴到光标所在位置的下一行。

---

## 3. 删除

```text
dd
```

作用：

> 删除光标所在的一整行。

删除多行：

```text
5dd
```

作用：

> 从当前行开始删除五行。

---

## 4. 撤销

```text
u
```

作用：

> 撤销上一次操作。

它的作用类似于 Windows 中的：

```text
Ctrl + Z
```

# 八、vim 光标移动

## 1. 使用方向键

在正常模式下，可以使用键盘上的方向键移动光标：

```text
↑  ↓  ←  →
```

---

## 2. 使用字母键移动

vim 中也可以使用以下按键移动光标：

| 按键 | 作用 |
| --- | --- |
| `h` | 向左移动 |
| `j` | 向下移动 |
| `k` | 向上移动 |
| `l` | 向右移动 |

当前阶段先掌握方向键即可，之后可以逐渐熟悉 `h`、`j`、`k`、`l`。

---

## 3. 跳转到文件末行

在正常模式下输入：

```text
G
```

作用：

> 跳转到文件的最后一行。

---

## 4. 跳转到文件首行

在正常模式下输入：

```text
gg
```

作用：

> 跳转到文件的第一行。

---

## 5. 跳转到指定行

例如：

```text
20G
```

作用：

> 跳转到第 20 行。

操作方式：

```text
目标行数字 + Shift + G
```

例如：

```text
10G
```

表示跳转到第 10 行。

# 九、vim 文件内容查找

## 1. 查找指定单词

在正常模式下输入：

```text
/关键字
```

例如：

```text
/Linux
```

然后按下回车，即可查找文件中出现的 `Linux`。

---

## 2. 查找下一个匹配内容

查找到一个内容后，输入：

```text
n
```

作用：

> 查找下一个匹配的内容。

查找流程：

```text
正常模式
    ↓
/关键字
    ↓
回车
    ↓
n查找下一个
```

# 十、vim 显示和取消行号

## 1. 设置行号

在正常模式下输入：

```text
:set nu
```

作用：

> 显示文件行号。

其中：

```text
nu = number
```

---

## 2. 取消行号

在正常模式下输入：

```text
:set nonu
```

作用：

> 取消显示文件行号。

其中：

```text
nonu = no number
```

# 十一、vim 基本操作示例

## 1. 创建并编辑文件

在终端输入：

```bash
vim hello.txt
```

进入 vim 后：

1. 默认进入正常模式。
2. 按下 `i` 进入插入模式。
3. 输入以下内容：


4. 按下 `Esc` 返回正常模式。
5. 输入 `:wq`。
6. 按下回车保存并退出。

## 2. 查看文件内容

退出 vim 后，在终端中输入：

```bash
cat hello.txt
```

即可查看文件内容。

---

## 3. 再次编辑文件

```bash

```

进入文件后，可以练习：

- 添加文本
- 删除一行
- 复制一行
- 粘贴一行
- 查找单词
- 设置行号
- 跳转到首行和末行
- 保存并退出

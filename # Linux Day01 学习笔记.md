# 一、Linux简介

## 1. Linux是什么？

Linux 是一种开源的操作系统内核。

它最早由 **Linus Torvalds（林纳斯·托瓦兹）** 于 1991 年创建。

Linux 具有：

- 开源
- 稳定
- 安全
- 高性能
- 可定制

等特点。

目前 Linux 已经广泛应用于服务器、嵌入式设备、云计算等领域。

---

# 二、Linux的好处

## 1. 开源免费

Linux 的源代码公开，任何人都可以查看、修改和优化。

优点：

- 降低使用成本
- 方便二次开发
- 社区资源丰富

---

## 2. 稳定性高

Linux 系统可以长时间运行而不需要频繁重启。

因此大量服务器选择 Linux 作为操作系统。

---

## 3. 安全性高

Linux 用户权限管理完善。

相比一些系统：

- 病毒较少
- 权限控制严格
- 系统更加安全

---

## 4. 可定制性强

用户可以根据需求：

- 修改系统
- 安装不同软件
- 定制系统环境

因此非常适合开发者使用。

---

# 三、Linux的应用场景

Linux 主要应用在三个领域：

---

# 1. 个人桌面领域

Linux 可以作为个人电脑操作系统。

例如：

- Ubuntu Desktop
- Linux Mint

应用：

- 编程开发
- 软件开发环境
- 日常办公

特点：

相比 Windows，Linux 更适合程序员进行开发工作。

# 2. 服务器领域（最主要）

服务器是 Linux 最重要的应用领域。

目前大量：

- 网站服务器
- 云服务器
- 数据中心
- 大型企业服务器

都使用 Linux。

原因：

- 稳定
- 安全
- 高性能
- 资源占用低

例如：

互联网公司的后台服务器大量运行 Linux。

---

# 3. 嵌入式领域

Linux 广泛应用于嵌入式设备。

例如：

- 智能机器人
- 智能家居
- 工业设备
- 自动驾驶设备

常见：

- 嵌入式 Linux
- Android 系统

对于未来机器人方向：

# 七、VMware 和 Linux 的关系

## VMware是什么？

VMware 是虚拟机软件。

作用：

可以在 Windows 系统中创建一个虚拟电脑。

例如：

Windows电脑

↓

VMware

↓

CentOS Linux虚拟机


这样不用安装双系统，也可以学习 Linux。

---

# 九、Linux磁盘分区

安装 CentOS 时学习了三个主要分区：

## 1. / 根目录

Linux最重要目录。

存放系统文件。

---

## 2. swap交换分区

# 十、VMware网络连接模式

VMware提供三种主要网络模式：

---

# 1. 桥接模式（Bridge）

特点：

虚拟机和主机处于同一个局域网。

例如：

电脑：

192.168.1.10

虚拟机：

192.168.1.20


优点：

- 可以被局域网其他设备访问

缺点：

- 占用真实网络IP

---

# 2. NAT模式

特点：

虚拟机通过主机访问互联网。

结构：

虚拟机

↓

VMware虚拟网络

↓

主机

↓

互联网


优点：

- 配置简单
- 最常使用

---

# 3. 仅主机模式（Host Only）

特点：

只能和主机通信。

不能访问外网。

适合：

- 测试环境
- 网络实验

---

# 十一、虚拟机管理

## 1. 克隆虚拟机

作用：

复制已有虚拟机。

优点：

快速创建多个相同环境。

例如：

创建多台Linux服务器进行测试。

---

## 2. 快照功能

作用：

保存当前虚拟机状态。

例如：

安装软件前：

创建快照。

如果出现问题：

可以恢复到之前状态。

---

## 3. 虚拟机迁移

作用：

将虚拟机移动到其他电脑。

需要迁移：

- 虚拟机文件
- 配置文件

---

## 4. 删除虚拟机

删除：

- 虚拟机配置
- 虚拟磁盘文件

释放电脑空间。
